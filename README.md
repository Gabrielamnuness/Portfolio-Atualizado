# Portfólio Gabriela Monteiro

Projeto profissional em React + TypeScript + Tailwind CSS.

## Rodar
```bash
npm install
npm run dev
```

## Personalizar
- Foto: substitua `public/gabriela.jpg`
- Currículo: substitua `public/curriculo.pdf`
- Dados pessoais/projetos: edite `src/data/site.ts`
